## Projeto da disciplina de Fundamentos de Sistemas Web
O Projeto da disciplina consiste na criação de uma plataforma online para um petshop.
Os produtos disponíveis estão divididos em 3 categorias: acessórios (roupas, brinquedos, camas, etc...), rações não perecíveis e produtos de higiene e limpeza (tapete higiênico, fraldas, etc..).
Quanto aos serviços oferecidos pelo petshop são: banho e tosa com ou sem tele-busca.
